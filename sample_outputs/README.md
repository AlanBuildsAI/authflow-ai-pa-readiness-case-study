# Sample Outputs

This folder contains recruiter-safe synthetic examples of AuthFlow AI / Plenara outputs.

## Safety

- No real patient data
- No PHI
- No live payer data
- No production clinical use

These examples are included only to show the structure of the workflow output and the type of readiness information the prototype can produce.

## Files

- `readiness_response_sample.json` — synthetic example response showing readiness status, blocker counts, review-required fields, confidence summary, and safety notice.
